# Real-Time Weather Checker

Get your country/city weather data (temperature and short description) by calling real-time weather API

## Languages
- JavaScript, HTML, CSS

## Methods
- <strike>XMLHttpRequest()</strike> to fetch()
- .value
- .open('GET', url)
- .responseType
- .send()
- if statement
- .status === 200
- JSON.parse()
